# App de Salud y Primeros Auxilios 🏥

Aplicación móvil desarrollada con MIT App Inventor enfocada en educar a las personas para actuar de manera adecuada en situaciones de emergencia.

## 🎯 Objetivo

Brindar información clara y accesible para que cualquier usuario pueda identificar y responder correctamente ante eventos de riesgo para la salud.

## 🚑 Funcionalidades

* Guía básica de primeros auxilios
* Identificación de síntomas de infarto
* Reconocimiento de signos de derrame cerebral
* Instrucciones para aplicar vendajes
* Recomendaciones para actuar en situaciones de emergencia

## 🏆 Logro

Segundo lugar en concurso estatal de creatividad e innovacion tecnologica

## 📱 Enfoque

Aplicación diseñada con una interfaz sencilla e intuitiva, pensada para ser utilizada por cualquier persona en momentos críticos.
